# YYTextFix

YYTextFix框架其实就是YYText，只不过YYText框架不更新了，为了修复和自己使用YYText方便，所以上传了这个YYTextFix。


## 安装

```
pod 'YYTextFix'
```

## 使用

使用见[YYText](https://github.com/ibireme/YYText)框架


## 协议
